# Avaliação de desenvolvimento Angular
### Concert Technologies
## Quick Start
### Backend
- Instalar dependência
```
npm install
```
- Startando o backend, rode no cmd:
```
json-server --watch db.json --port 3001
```
### Frontend
- Instalar dependência
```
npm install
```
- Startando servidor para execução da aplicação em Angular
```
ng serve
```
